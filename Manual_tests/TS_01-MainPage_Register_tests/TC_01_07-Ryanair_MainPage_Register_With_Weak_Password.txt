Title - TC_01_07-Ryanair_MainPage_Register_With_Weak_Password
Description - Test case that tests registration with weak password

Pre-conditions
  1) Operating system at least Windows 10 or macOS Sonoma 14.3.1
  2) Internet connection available
  3) Open website https://www.ryanair.com/cz/cs
  4) Mouse, keyboard and display available
  5) Refuse cookies by clicking the button "Ne děkuji"
  6) Click button "Registrovat" in the upper-right bar

Steps
  1) Put "passwd" into the email field
  2) Click button "Vytvořit účet"

Expected result
  Label and border of the password field should have error color

Version - 1.0
